# Ruflo Concept Index

This is a selected Ruflo reference for foraging. It is intentionally small so
Colony can search by ideas without importing Ruflo as a file source.

Ruflo's useful concepts for Colony:

- Outcome learning: store task outcomes, verification results, failures, and feedback as reusable learning signals.
- Pattern memory: search prior observations, history, and recallable patterns before starting a new lane.
- Trigger routing: classify prompts and route work to the right tool, agent, or workflow.
- Token budget: keep compact context, hydrate only needed bodies, and collapse repeated status noise.
- Sidecar runtime: run background workers as a service beside the main CLI.
- MCP bridge: expose memory, planning, and tool calls through a Model Context Protocol server.
- Goal planning: decompose plain-English goals into actions, preconditions, replanning, and progress state.
- Ready-work ranking: rank queued work by priority, capability fit, stale ownership, and evidence.
- AgentDB and RuVector: use vector memory, HNSW search, embeddings, and graph RAG for semantic recall.
- Federation: coordinate agents across machines through identity, trust, and audit records.
