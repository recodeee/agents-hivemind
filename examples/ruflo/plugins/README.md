# Ruflo Plugin Catalog Concepts

Selected Ruflo plugin ideas worth porting as concepts:

- Plugin registry and plugin marketplace: discover capabilities by name, not by copied folders.
- Tool catalog: list MCP tools, commands, skills, and sidecar actions in one searchable index.
- AgentDB plugin: persistent pattern memory, semantic route, hierarchical recall, and causal edges.
- RuVector plugin: vector search, embeddings, graph RAG, and fast nearest-neighbor lookup.
- Federation plugin: cross-installation agent collaboration with trust scoring and audit trails.
- Goals plugin: goal planning, plan trees, progress tracking, and adaptive replanning.
- Cost tracker plugin: token budget tracking, compaction, and usage feedback.
